Jawaban no 1

Website adalah sekumpulan halaman web yang saling terhubung dan dapat diakses melalui internet. Setiap halaman web memiliki konten, seperti teks, gambar, audio, atau video, yang dapat diakses oleh pengguna menggunakan perangkat yang terhubung ke internet melalui browser. Website beroperasi melalui kolaborasi kompleks antara web server, domain, dan hosting. Web server, yang menyimpan dan mengelola data, merespons permintaan pengguna dengan menyajikan halaman web. Domain berfungsi sebagai alamat unik yang mengarahkan ke server melalui DNS, sementara hosting adalah tempat penyimpanan file dan data website. Saat seseorang mengakses domain, permintaan diteruskan dari DNS ke server hosting, yang kemudian mengirimkan data halaman web ke browser pengguna. Dengan demikian, web server, domain, dan hosting bekerja bersama untuk menyediakan akses dan pengalaman pengguna dalam menjelajahi website.
